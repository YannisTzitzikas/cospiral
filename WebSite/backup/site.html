CoSpi Concentric Spiral

Description

Concentric Spiral is a novel placement algorithm, designed to visualize large datasets efficiently in a spiral form, representing values as geometrical objects. The algorithm has linear time complexity (when the dataset is sorted), minimal memory requirements and it's a subject to several variations and implementations. In general, CoSpi represents values in an a geometrical form, calculating their corresponding coordinates in a given canvas in a way that the builings are placed in a cyclic form, starting from the biggest value, making sure that the objects never collide, resulting in a gemotrical dataset representation a the shape of a spiral.

Examples

This section contains basic examples of the CoSpi tool. For every discrete example, the input file is given and the visualization is produced on two versions: As raster image and as SVG image. Raster images are not scalable images, but they provide a general look of how the algorithm works for the corresponding dataset. SVG files are scalable and are produced by the tool to extend the actions that can be applied to the plot, such as searching and traversing the data.
